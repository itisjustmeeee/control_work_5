Вариант 30: мини-игра "Угадай число"

локальный сервер:  http://localhost:3000

Функционал:
простой интерфейс с кнопками: "угадать" и "начать игру". Без "начать игру" будет невозможно играть, так как ничего не будет запущено.
Также есть поле ввода числа.
При каждой попытке будет писать подсказки для угадывания числа, больше ли текущее или меньше + колчество попыток

необходимо клонировать репозиторий себе и запустить (node App.js + npm install espress(Если он не установлен)). Потом открыть локальный сервер
